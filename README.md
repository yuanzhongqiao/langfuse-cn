<h2 tabindex="-1" dir="auto"><a id="user-content-what-is-langfuse" class="anchor" aria-hidden="true" tabindex="-1" href="#what-is-langfuse"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">朗福斯是什么？</font></font></h2>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Langfuse 是一个针对基于 LLM 的应用程序的开源可观测性和分析解决方案。</font><font style="vertical-align: inherit;">它主要面向生产用途，但一些用户也将其用于 LLM 应用程序的本地开发。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Langfuse 专注于基于法学硕士构建的应用程序。</font><font style="vertical-align: inherit;">最近出现了许多新的抽象和常见的最佳实践，例如代理、链式提示、基于嵌入的检索、LLM 对 REPL 和 API 的访问。</font><font style="vertical-align: inherit;">这些使得应用程序更加强大，但对于开发人员来说也是不可预测的，因为他们无法完全预测变化如何影响其应用程序的质量、成本和总体延迟。</font><font style="vertical-align: inherit;">因此，Langfuse 有助于监控和调试这些应用程序。</font></font></p>
